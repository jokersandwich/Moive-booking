# Moive-booking
假想有一个电影爱好者自己开的影院，每天播放豆瓣电影Top250，这是它的订票小程序（demo）。


## 说明

这是一个简单的上手项目，用于熟悉小程序基本用法。

豆瓣电影接口来自Github社区 [@zec](https://github.com/zce)。

设计图来自Uplabs社区 [@Ayush Jangra](https://www.uplabs.com/ayushjangra)。
