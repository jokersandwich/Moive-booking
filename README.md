# Moive-booking
假想有一个电影爱好者自己开的影院，每天播放豆瓣电影Top250，这是它的订票小程序（demo）。


## 说明

豆瓣电影接口来自Github [@zec](https://github.com/zce)。

设计原图来自Uplabs [@Ayush Jangra](https://www.uplabs.com/ayushjangra)。
